# 해외 헬스장 API 비교 조사

## 조사 일자: 2026-01-13

## 개요
글로벌 확장을 위해 해외 헬스장/피트니스 센터 정보를 제공할 수 있는 API 후보를 조사했습니다.

---

## API 비교표

| API | 지역 커버리지 | 무료 티어 | 비용 모델 | 주요 기능 |
|-----|-------------|----------|----------|----------|
| **Google Places API** | 글로벌 (200+ 국가) | 월 $200 크레딧 | 사용량 기반 | 검색, 상세, 사진, 리뷰 |
| **Yelp Fusion API** | 미국, 캐나다, EU, UK | 월 5,000 요청 | 무료/유료 | 리뷰, 평점, 카테고리 |
| **Foursquare Places** | 글로벌 | 월 100,000 요청 | 무료/유료 | 위치, 카테고리, 인기도 |
| **TripAdvisor Content** | 글로벌 (관광지 중심) | 제한적 | 별도 계약 | 리뷰, 사진 |

## 상세 분석

### 1. Google Places API

**장점:**
- 가장 넓은 커버리지 (200+ 국가)
- 정확한 위치 데이터
- 풍부한 부가 정보 (영업시간, 사진, 리뷰)
- 한국어/영어 등 다국어 지원

**단점:**
- 비용이 빠르게 증가할 수 있음
- 캐싱 제한 (30일)
- 결과에 Google 로고 표시 필수

**비용:**
- Place Search: $32 / 1,000 요청
- Place Details: $17 / 1,000 요청
- 월 $200 크레딧으로 약 6,000회 상세 조회 가능

**적합도:** ⭐⭐⭐⭐⭐ (5/5)

---

### 2. Yelp Fusion API

**장점:**
- 리뷰 데이터가 풍부
- 실제 사용자 평점
- 깔끔한 API 설계

**단점:**
- 아시아 지역 데이터 부족 (한국 제외)
- 일부 국가 미지원
- 비즈니스용 별도 약관

**비용:**
- 무료: 월 5,000 요청
- 유료: 볼륨에 따라 협의

**적합도:** ⭐⭐⭐ (3/5) - 미국/EU 타겟 시 고려

---

### 3. Foursquare Places API

**장점:**
- 높은 무료 쿼터 (월 100,000회)
- 위치 정확도 높음
- 카테고리 분류 상세

**단점:**
- 리뷰 데이터 상대적으로 적음
- 최근 API 구조 변경

**비용:**
- 무료: 월 100,000 요청
- Developer+: 월 $499

**적합도:** ⭐⭐⭐⭐ (4/5) - 비용 효율적

---

### 4. TripAdvisor Content API

**장점:**
- 여행자 중심 풍부한 리뷰
- 사진 품질 높음

**단점:**
- 헬스장보다 호텔/관광지 특화
- API 접근 제한적 (파트너십 필요)
- 비용 불투명

**적합도:** ⭐⭐ (2/5) - 범위 제한

---

## 추천

### 1순위: Google Places API
- **이유:** 가장 넓은 커버리지, 다국어 지원, 안정성
- **적용 시기:** Phase 4+ (글로벌 런칭 시)
- **예상 비용:** 초기 무료 ($200 크레딧), 이후 월 $50-200

### 2순위: Foursquare Places API
- **이유:** 높은 무료 쿼터, 비용 효율적
- **적용 시기:** MVP 또는 비용 제약 시
- **예상 비용:** 대부분 무료 범위 내
