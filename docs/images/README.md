# 이미지 저장소

이 폴더는 REHAB 앱의 UI/UX 참고 이미지를 저장하는 곳입니다.

## 이미지 파일

### 재활 코스 생성 플로우
- **파일명**: `rehab-course-generation-flow.png` (또는 `.jpg`, `.webp`)
- **설명**: 헬스장 검색부터 맞춤형 재활 코스 생성까지의 전체 플로우를 보여주는 이미지
- **사용 위치**: `docs/DESIGN_SYSTEM.md`의 "UI/UX 참고 이미지" 섹션

### REHAB 앱 전체 디자인 (6개 화면)
- **파일명**: `rehab-app-design-6-screens.png` (또는 `.jpg`, `.webp`)
- **설명**: REHAB 앱의 전체적인 UI/UX 디자인을 보여주는 6개의 스마트폰 화면 (홈, 헬스장 찾기, 코스 생성, 루틴, 헬스장 상세)
- **색상 스키마**: 다크 퍼플/블랙 배경 + 화이트 텍스트 + 퍼플 액센트
- **사용 위치**: `docs/DESIGN_SYSTEM.md`의 "UI/UX 참고 이미지" 섹션 및 색상 팔레트 정의

## 이미지 추가 방법

1. 이미지 파일을 이 폴더(`docs/images/`)에 저장합니다.
2. `docs/DESIGN_SYSTEM.md` 또는 관련 문서에 이미지 참조를 추가합니다.
3. Markdown 형식: `![이미지 설명](./images/파일명.png)`

## 권장 이미지 포맷

- **포맷**: PNG, JPG, WebP
- **해상도**: 최소 1920px 너비 (고해상도 디스플레이 대응)
- **최적화**: 파일 크기 최소화 (WebP 권장)
- **접근성**: Alt 텍스트 필수
